Este proyecto está compuesto de 2 carpetas(Matrices y Scripts), tres archivos comprimidos
(Elevation, WORLDCLIM.part1y WORLDCLIM.part2) y un archivo .phy que contiene la filogenia para 1418 especies 
analizadas en este trabajo.

La carpeta de Matrices contiene 3 archivos csv:
1. Matriz_9_31-08-2022, el cual contiene las especies con 13 caracteres y sus estados de caracter (Ver readme.Matriz para mayor comprension);
2.Matriz_9_trait_clim_12-10-2022, esta matriz contiene la misma información que la anterior, con
la difernecia de que además tiene incluída la información climática para MAT, MAP y PDQ
3.sp_list, contiene el listado especies para crear la filogenia.

El archivo comprimido Elevation contiene información sobre elevación y los archivos WORLDCLIM.part1y WORLDCLIM.part2
tienen información climática de de 19 variables climáticas.
